## 3. Curriculum (/curriculum)

**Purpose**: Detail the 10-week structured learning program

**Sections**:
1. Hero/Header
2. Overview
3. Weekly Breakdown (10 weeks)
4. Learning Outcomes
5. Prerequisites
6. How to Enroll

**Data Requirements**:

### Hero Section
- **Headline**: "10-Week Quantitative Finance Curriculum"
- **Subheadline**: "A structured program designed to build expertise from fundamentals to advanced applications"

### Overview
- **Description** (copy):
  "Our comprehensive 10-week curriculum covers essential quantitative finance topics, from mathematical foundations to algorithmic trading strategies. Each week builds upon previous concepts, culminating in a capstone project."

### Weekly Breakdown
- **Week 1**: Introduction to Quantitative Finance
  - Topics: Overview of quant finance, career paths, mathematical prerequisites
  - Deliverables: Market research assignment
  
- **Week 2**: Financial Markets & Instruments
  - Topics: Equity, fixed income, derivatives, market microstructure
  - Deliverables: Instrument analysis project
  
- **Week 3**: Time Series Analysis
  - Topics: ARIMA, GARCH, volatility modeling, stationarity
  - Deliverables: Time series forecasting exercise
  
- **Week 4**: Portfolio Theory & Optimization
  - Topics: Modern Portfolio Theory, Markowitz optimization, risk-return tradeoffs
  - Deliverables: Portfolio construction case study
  
- **Week 5**: Options Pricing Models
  - Topics: Black-Scholes, binomial models, Greeks, volatility surfaces
  - Deliverables: Options pricing calculator
  
- **Week 6**: Risk Management
  - Topics: VaR, CVaR, stress testing, risk metrics
  - Deliverables: Risk analysis report
  
- **Week 7**: Algorithmic Trading Strategies
  - Topics: Market making, statistical arbitrage, execution algorithms
  - Deliverables: Strategy backtesting project
  
- **Week 8**: Machine Learning in Finance
  - Topics: Regression, classification, feature engineering, model validation
  - Deliverables: ML trading signal development
  
- **Week 9**: Quantitative Research Methods
  - Topics: Factor models, alpha research, signal generation
  - Deliverables: Research paper draft
  
- **Week 10**: Capstone Project
  - Topics: Integration of all concepts, presentation skills
  - Deliverables: Final project presentation

### Learning Outcomes
- **Outcomes** (bullet list):
  - Master core quantitative finance concepts and models
  - Develop proficiency in Python/R for financial analysis
  - Build and backtest trading strategies
  - Understand risk management frameworks
  - Create a portfolio-ready project

### Prerequisites
- **Requirements** (copy):
  - Strong foundation in calculus, linear algebra, and statistics
  - Basic programming experience (Python preferred)
  - Interest in financial markets
  - Commitment to 10-week program

### How to Enroll
- **CTA Button**: "Apply for Next Cohort"
- **Link**: /join
