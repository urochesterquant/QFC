Spring 2026 Curriculum

A structured 10-week program covering essential quantitative finance topics from fundamentals to advanced applications

Program Overview

Our comprehensive 10-week curriculum covers essential quantitative finance topics, from mathematical foundations to algorithmic trading strategies. Each week builds upon previous concepts, culminating in a capstone project showcase.

Week 1 February 6

Introduction to Quantitative Finance

Goals

  • Understand the landscape of quantitative finance and career paths
  • Review mathematical prerequisites (calculus, linear algebra, statistics)
  • Set up development environment (Python/R, data sources)
  • Introduction to financial markets and instruments

Prerequisites

  • Strong foundation in calculus and linear algebra
  • Basic programming experience (Python preferred)
  • Interest in financial markets

What You'll Build

  • Market research assignment analyzing current market conditions
  • Development environment setup with required libraries
  • Initial portfolio of financial data sources

Recommended Resources

  • "Quantitative Finance" by Paul Wilmott - Chapters 1-3
  • Python for Finance (Coursera) - Setup and basics
  • FRED Economic Data - Data source exploration
Week 2 February 13

Backtesting Fundamentals

Goals

  • Understand backtesting methodology and best practices
  • Learn to avoid common backtesting pitfalls (lookahead bias, survivorship bias)
  • Implement a basic backtesting framework
  • Analyze strategy performance metrics (Sharpe ratio, drawdowns)

Prerequisites

  • Week 1 completion
  • Basic Python programming
  • Understanding of time series data

What You'll Build

  • Simple backtesting engine using historical data
  • Performance analytics dashboard
  • Strategy comparison framework

Recommended Resources

  • "Advances in Financial Machine Learning" by Marcos L√≥pez de Prado - Backtesting chapter
  • Zipline Documentation - Open-source backtesting library
  • QuantStart Backtesting Guide - Best practices
Week 3 February 20

Statistical Arbitrage

Goals

  • Understand pairs trading and mean reversion strategies
  • Learn cointegration and correlation analysis
  • Implement statistical arbitrage signals
  • Risk management for stat arb strategies

Prerequisites

  • Week 2 completion
  • Understanding of correlation and regression
  • Basic backtesting knowledge

What You'll Build

  • Pairs trading strategy with cointegration testing
  • Signal generation system for stat arb
  • Risk-adjusted performance analysis

Recommended Resources

  • "Pairs Trading" by Ganapathy Vidyamurthy - Core concepts
  • QuantInsti Statistical Arbitrage Course - Practical implementation
  • Python statsmodels library - Cointegration testing
Week 4 February 27

Alpha Factors

Goals

  • Learn to identify and construct alpha factors
  • Understand factor analysis and signal processing
  • Implement factor-based trading strategies
  • Evaluate factor performance and decay

Prerequisites

  • Week 3 completion
  • Understanding of statistical analysis
  • Familiarity with financial data structures

What You'll Build

  • Alpha factor research framework
  • Factor performance evaluation system
  • Multi-factor model implementation

Recommended Resources

  • "Quantitative Trading" by Ernest Chan - Factor construction
  • WorldQuant WebSim - Factor research platform
  • QuantConnect Alpha Streams - Factor marketplace
Week 5 March 6

Options Pricing & Greeks

Goals

  • Master Black-Scholes and binomial option pricing models
  • Understand and implement the Greeks (Delta, Gamma, Theta, Vega, Rho)
  • Learn volatility surfaces and implied volatility
  • Implement options trading strategies

Prerequisites

  • Week 4 completion
  • Understanding of stochastic processes
  • Calculus and probability theory

What You'll Build

  • Options pricing calculator (Black-Scholes & binomial)
  • Greeks calculation and visualization tool
  • Volatility surface analysis system

Recommended Resources

  • "Options, Futures, and Other Derivatives" by John Hull - Chapters 11-19
  • QuantLib Python - Options pricing library
  • CBOE Options Education - Market mechanics

Week 6 March 20

Volatility Modeling

Goals

  • Understand volatility clustering and stylized facts
  • Implement GARCH models for volatility forecasting
  • Learn stochastic volatility models
  • Apply volatility models to risk management

Prerequisites

  • Week 5 completion
  • Understanding of time series analysis
  • Knowledge of maximum likelihood estimation

What You'll Build

  • GARCH volatility forecasting model
  • Volatility surface construction tool
  • Risk-adjusted portfolio analysis

Recommended Resources

  • "Analysis of Financial Time Series" by Ruey Tsay - GARCH models
  • ARCH Python Package - Volatility modeling
  • V-Lab Volatility Research - Academic resources
Week 7 March 27

Portfolio Optimization

Goals

  • Master Modern Portfolio Theory and Markowitz optimization
  • Implement risk parity and minimum variance portfolios
  • Learn factor-based portfolio construction
  • Understand transaction costs and constraints

Prerequisites

  • Week 6 completion
  • Understanding of covariance and correlation matrices
  • Basic optimization techniques

What You'll Build

  • Portfolio optimization engine (Markowitz, risk parity)
  • Efficient frontier visualization
  • Multi-asset portfolio rebalancing system

Recommended Resources

  • "Portfolio Selection" by Harry Markowitz - Original paper
  • PyPortfolioOpt Library - Python implementation
  • "Advances in Portfolio Optimization" by Bernd Scherer - Advanced topics
Week 8 April 3

Risk Management

Goals

  • Understand Value at Risk (VaR) and Conditional VaR (CVaR)
  • Implement stress testing and scenario analysis
  • Learn risk metrics and limit frameworks
  • Apply risk management to trading strategies

Prerequisites

  • Week 7 completion
  • Understanding of probability distributions
  • Knowledge of portfolio theory

What You'll Build

  • VaR and CVaR calculation system
  • Stress testing framework
  • Risk dashboard and reporting tool

Recommended Resources

  • "Risk Management and Financial Institutions" by John Hull - VaR methods
  • QuantLib Risk Analytics - Risk calculation library
  • FRTB Documentation - Regulatory risk frameworks
Week 9 April 10

Recruiting & Career Development

Goals

  • Understand quant finance career paths and roles
  • Learn interview preparation strategies
  • Build a portfolio-ready project presentation
  • Network with industry professionals and alumni

Prerequisites

  • Week 8 completion
  • Completed project from previous weeks
  • Resume/CV draft

What You'll Build

  • Professional project presentation
  • Portfolio-ready GitHub repository
  • Technical interview preparation materials

Recommended Resources

  • QuantNet Career Guide - Industry insights
  • "Heard on the Street" by Timothy Crack - Interview prep
  • LinkedIn Quant Finance Groups - Networking
Week 10 April 17

Capstone Showcase

Goals

  • Present capstone projects to peers and industry professionals
  • Receive feedback and iterate on projects
  • Celebrate achievements and network
  • Plan next steps for continued learning

Prerequisites

What You'll Build

  • Integration of concepts from all previous weeks

  • Professional presentation and documentation
