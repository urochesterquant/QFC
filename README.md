# Quantitative Finance Club Website

## Design System

### Colors
- **Primary**: Meliora Blue (#021BC3)
- **Accent 1**: Quantum Cobalt (#003EFF)
- **Accent 2**: Arpeggio Azure (#0066FD)

### Typography
- **Font Family**: Inter
- **Scale**: 12px to 60px with consistent line heights

### Spacing
- Consistent spacing scale from 4px to 128px
- Section padding: 128px (desktop) / 64px (mobile)

### Components
- Buttons (Primary, Secondary, Large variants)
- Cards with hover effects
- Navigation bar with smooth transitions
- Footer with organized links

## Placeholders

The following files need to be replaced with actual assets:
- `placeholder-club-logo.svg` - Replace with actual club logo
- `placeholder-ur-mark.svg` - Replace with University of Rochester mark
- Hero background imagery can be added to `.hero-background` class

## Usage

1. Open `index.html` in a web browser
2. The theme toggle in the navigation bar switches between light and dark modes
3. Navigation links smoothly scroll to respective sections

## Browser Support

Modern browsers with support for:
- CSS Custom Properties (Variables)
- CSS Grid
- Flexbox
- Intersection Observer API
- Smooth scrolling

## Customization

All design tokens are defined in CSS variables at the top of `styles.css`. Modify these to adjust:
- Colors
- Typography scale
- Spacing
- Border radius
- Shadows
- Transitions
